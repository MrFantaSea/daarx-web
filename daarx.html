DAARX

The one designed to keep. A private-money protocol target, not a live network or available asset.

Hardware interest → How it works ↓

the supply

The current consensus design targets π × φ base units — about 5.083 septillion — reached exactly by an integer-only hundred-year schedule and then zero emission. The table is implemented and testable, but it is not a promise about an unlaunched asset and remains subject to the final consensus review and genesis process.

π × φcurrent consensus design target

The implementation uses a deterministic integer table that can be recomputed. It becomes a live supply rule only after final review, genesis, and public launch.

the privacy

DAARX targets a full-chain anonymity set rather than a small ring and uses hidden amounts with spent-tag checks. That is the cryptographic design and implementation target, not a guarantee of anonymity: metadata, endpoint compromise, implementation flaws, network observation, and novel cryptanalysis remain threats that require public testing.

Fees are designed to use a short public ladder to reduce fee fingerprinting. The ingress policy and end-to-end miner path still require launch-level integration evidence.

the limits

The privacy layer is new cryptography built on reviewed dependencies. It is tested, not proven, and the project has not completed the independent review and long public soak required for a launch claim. A security program is planned but no bounty is currently open.

Status — July 14, 2026: implementation and audit work are active. The public testnet package, integrated testnet miner, release wallet, independent review evidence, operational runbooks, and final whitepaper are not released. No release date is promised.
Mining is the intended distribution path.

The launch design has no premine, presale, or coin sale. That policy becomes meaningful only if a reviewed network launches. Today there is no public testnet mining release and rig submissions are non-binding market research, not orders.
